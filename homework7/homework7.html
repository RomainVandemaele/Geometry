Homework 7

Control explanation :

You can place at most 100 points. When all of your points are placed, you can press "enter" to display the complex hull.

Algorithm explanation :

I implemented the Jarvis March algorithm with some modifications to handle special cases. If several point are possible for the leftmost one, the choosen one is the one with the lowest y-coordinate. This choice comes from the observation that all the points at the leftmost will be included in the convex hull so the higher or the lowest can be choosen. For choosing the next point, if 2 points have the same angle, the one which is the farest is choosen. Because if the closest was choosen first, the farest will be choosen next.

Remark : The coordnates of points are rounded (e.g : 123 => 120) s.t. it is easier to place colinear point.
